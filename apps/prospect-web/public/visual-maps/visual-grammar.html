Visual Alignment

Color = Status
Shape = Role
Position = Sequence

Containers stay slate. Meaning nodes get the color. Split the map after 9 important nodes.

Colors

Blue Evidence, input, read-only facts
Green Verified truth, safe confirmed state
Amber Human review, ambiguity, decision needed
Red Mutation, risk, stop, destructive action, bug
Purple External automation, AI, generated proposal
Slate Neutral container, UI shell, grouping, passive context

Shapes

PPerson Human, operator, client
Browser Prospect Web, Raycast UI
Storage Supabase, Messages DB, durable table
Queue Pending tasks, backlog, review queue
Document Generated receipt, evidence bundle
Rectangle Normal domain component
Bucket Broad grouping only, lightly styled

Layouts
