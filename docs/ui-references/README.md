# UI References

Reference buckets for Scouting Command UI work.

Use these files to preserve visual direction without mixing unrelated workflows into Scout Prep. Screenshots pasted in chat are recorded as references here; if exported to disk later, place them in the matching bucket using the suggested filenames.

## Buckets

- `scout-prep/` - selected-athlete task detail, operator actions, contact/notes/task surfaces.
- `set-meetings/` - calendar, head-scout schedule, set-meeting and view-set-meetings surfaces.
- `client-messages/` - inbox, recipient selection, message context, Messages.app handoff.
- `prospect-search/` - search results, athlete lookup, admin navigation.
- `shared-navigation/` - app shell, sidebar, tabs, task list/detail layout.
