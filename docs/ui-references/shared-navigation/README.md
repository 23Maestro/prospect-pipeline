# Shared Navigation UI References

## Direction

The Scouting Command shell should feel like Raycast x Asana:

- Left navigation for major side panels.
- Main task list for Scout Prep queue.
- Selected item detail pane for athlete/task work.
- Top-right filters/tabs for Scout Prep buckets.

## Use

- Asana's sidebar/header/main-pane/task-detail mental model.
- Raycast's fast command/action feel.
- Minimal operational copy.
- Status chips and due-state accents for life.

## Avoid

- Marketing language.
- Debug labels.
- Function names.
- Redundant explanatory subtitles.

- Asana task fields/detail pane: https://help.asana.com/s/article/task-fields
